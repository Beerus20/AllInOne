# Renderer Comparison Example

This example demonstrates how to compare different rendering backends:
- SDL2
- OpenGL
- MinilibX
- Vulkan

## Running

```bash
./renderer-comparison --renderer=SDL
./renderer-comparison --renderer=OpenGL
./renderer-comparison --renderer=MinilibX
./renderer-comparison --renderer=Vulkan
```

## Performance Metrics

The example will display FPS and render time for each backend.
