# Custom Plugin Example

This example shows how to create a custom plugin for the One framework.

## Structure

- `MyCustomPlugin.hpp` - Plugin header
- `MyCustomPlugin.cpp` - Plugin implementation
- `CMakeLists.txt` - Build configuration

## Building

See the root CMakeLists.txt for build instructions.

## Implementation Details

Refer to the plugin development guide in `docs/plugin-development.md`.
